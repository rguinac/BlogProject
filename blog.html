Phase 1 Reflection

October 27, 2020 by Rudy Guinac

Well we have finished Phase 1 of the Simplilearn course and while I still have a lot to learn using HTML, CSS, and JS, I've learned quite a bit. One thing is for sure, I still got to study on my own time as well.

One thing that I really liked about the Phase 1 portion of the course was having a bunch of examples that the teacher would work out so I can visually see how everything would work as they were being implemented.

Another aspect that I'm really happy about with course so far is that the teacher also uploads all the code from the sessions up on her Github so we can go back and refer to her code.

This is so useful since I can go back and check to see her work. This allows me to work out any potential problem I ran across while initially working on it during the session and see what I did wrong and how to fix it.

Phase 1

October 19, 2020 by Rudy Guinac

Today we started a whole new phase for the ILP training for TCS. The previous phase was more of a recap of how Java works, what you can do with it, and much more. It was great to be able to recap so much Java since the last time I worked with Java was back in my sophmore year of college.

Overall so far, I have enjoyed my time being at TCS and have enjoyed talking to a bunch of new people that have various backgrounds in coding. While I think my coding skills are kinda rusty in comparison, I'm still giving it all my effort.

I'm very curious how the next phase of the ILP training will go. We'll be working in a language that, while I do have some kind of knowledge, will be completely new to me.

Hopefully this phase will be just as interesting and engaging as the last phase.
